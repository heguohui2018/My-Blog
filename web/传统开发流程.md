  ## 传统开发流程
  策划 -\\> 交互 -\\> 视觉 -\\> 前端 -\\> 后端 -\\> 测试
  
  ## 前端职责
  
  视觉稿（配色图标距离空间） \\+ 交互稿（用户逻辑） = UI（用户界面）
  
  视觉稿 -\\> 页面制作
  
  交互稿 -\\> 页面逻辑开发
  
  ## 所需技能
  
  *   DOM (操作 HTML 及 CSS 的接口)
  *   JavaScript (定义页面互动)
  *   CSS (定义页面样式)
  *   HTML (定义页面结果)
  *   Photoshop (取图)
  
  切图:从设计稿中切除网页的素材并在代码中引入图片 (复杂的图片或者解决兼容问题)
